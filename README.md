# RecipeRandom

RecipeRandom is a SwiftUI iOS app for discovering and saving dinner ideas, generating new recipes, and managing a personal shopping list.

## Features
- Generate new recipe ideas (local-only)
- Save and unsave recipes
- Shopping list with saved-recipe ingredients + personal items
- Offline-first local persistence

## Requirements
- Xcode 15+
- iOS 17+

## Run
1. Open `RecipeRandom.xcodeproj`
2. Select an iOS Simulator
3. Build and Run

## Notes
This project is local-only and does not include network APIs or authentication yet.
