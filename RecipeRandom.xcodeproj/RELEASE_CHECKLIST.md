# Release Checklist (iOS)

## Pre-release QA
- [ ] Generate recipe → open detail → save/unsave works
- [ ] Saved list updates immediately
- [ ] Shopping list: derived + personal items work
- [ ] Persistence after relaunch
- [ ] App icon visible on device

## Project Settings
- [ ] Display Name set
- [ ] Bundle ID correct
- [ ] Version / Build updated
- [ ] iOS deployment target set (e.g., 16+)
- [ ] No critical warnings or errors

## App Store Connect
- [ ] App record created
- [ ] Metadata filled (name, subtitle, description, keywords)
- [ ] Category selected
- [ ] Support URL + Privacy Policy URL
- [ ] App Privacy: no data collected
- [ ] Screenshots uploaded

## Build & Submit
- [ ] Archive in Xcode
- [ ] Upload build to App Store Connect
- [ ] Build selected in version
- [ ] Submit for review
